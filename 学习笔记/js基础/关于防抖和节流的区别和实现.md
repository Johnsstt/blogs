## 防抖和节流的区别
### 防抖是...
我理解的防抖是，在事件被连续触发时，不进行处理，当连续触发停止时延迟处理。
### 而节流..
我理解的节流是，在事件被连续触发时，

## 乞丐版实现
```js
const container = document.getElementById('container')

// 防抖
function debounce(func, time) {
    let timer;
    return () => {
        clearTimeout(timer)
        timer = setTimeout(() => {
            func()
        }, time);
    }
}

// 节流
function throttle(func, time) {
    let timer;
    return () => {
        if (!timer) {
            timer = setTimeout(() => {
                func()
                timer = null
            }, time)
        }
    }
}

container.addEventListener('scroll', debounce(() => { console.log('a') }, 1000))
```
