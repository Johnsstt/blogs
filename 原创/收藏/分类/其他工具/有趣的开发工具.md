它具有以下功能：

```
评论（Disqus整合）
谷歌分析

图像优化器
GraphQL for json API
角色 - 管理员，审稿人，作者，读者
Markdown和RichText编辑器
搜索引擎优化
多语言支持（当前en，fr和pl）
```
